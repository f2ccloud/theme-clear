# Clear 主题

基于 Halo2.0 开发，将当前目录压缩为 zip 包后安装

基于 [LoveIt](https://github.com/dillonzq/LoveIt) Hugo 主题模版进行开发

初始化设置步骤如下：

- [x] 如果需要展示博客内容，需要在 Console 端添加 **文章 / 标签 / 分类**，然后将文章在设置里划分到对应的标签和分类中
- [x] 该主题需要配合 Halo 官方的评论组件、搜索组件的插件使用
